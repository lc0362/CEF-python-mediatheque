Mise en route du projet 
•	ouvrir le dossier dans VS Code
•	taper la commande .env\Scripts\activate
•	taper la commande cd src
•	taper la commande python manage.py runserver
•	cliquer sur le lien http://127.0.0.1:8000/

Mise en place des tests
•	taper commande python manage.py test bibliothecaire.tests
